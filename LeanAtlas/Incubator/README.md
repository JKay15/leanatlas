# LeanAtlas/Incubator

Here is the **incubation area**:

- `Seeds/`: candidate tool (retrievable, on-demand import, recyclable).
- `Archive/`: Isolated recycling area (can be rolled back).
- `External/`: Entrance to external bundles (enter incubation first, run gate to be promoted).

V0 constraint: GC only changes `tools/index/gc_state.json` and does not move files.
