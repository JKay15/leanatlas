# mk_queue_mg1_lindley (fixture)

This fixture targets the **underlying recurrence** in queueing theory (Lindley's recursion).

It is intentionally modest in Lean difficulty but realistic in structure:
- recursive definition (`Nat → ℝ`)
- induction proof over `Nat`
- basic real inequality reasoning

In later eval packs we can extend this to Pollaczek–Khinchine or stability conditions.
